# MS Emergência 190/193 - QA 

Este repositório contém o planejamento, documentação, execução e automação de testes realizados sobre o aplicativo mobile MS Emergência 190/193.

## Objetivos

- Planejar os testes do aplicativo

- Criar casos de teste

- Executar testes funcionais

- Realizar testes exploratórios

- Documentar defeitos

- Produzir relatórios de execução

- Automatizar cenários críticos

## Funcionalidades Testadas

- Login

- Recuperação de senha

- Home

- Seleção de Ocorrências

- Questionário

- Geolocalização

- Captura de Imagens

- Envio da Solicitação

- Perfil
